# VISNEK!

Controls:
  * i - Enter input mode
  * esc - Enter control mode

  * h - Left
  * j - Down
  * k - Up
  * l - Right
---
Notes:
  * Symbol looking like :: means space
  * You don't have to type symbols that look like *
